Это тестовый пост для проверки визуала темы. Здесь я проверяю как выглядят различные элементы форматирования текста в темной теме блога.

Форматирование

Давайте посмотрим на различные варианты форматирования текста:

  • Жирный текст для выделения важных моментов
  • Курсив текст для акцентов
  • Списки для структурирования информации
  • Цитаты для выделения важных мыслей
Это цитата. Текст цитаты для проверки типографики темы. Цитаты отлично подходят для выделения важных мыслей или высказываний других авторов.

Код

Одна из важнейших функций технического блога - красиво отображать код. Вот пример bash команды:

echo "Hello, world!"
git status
npm install

А здесь пример инлайн кода: const greeting = "Hello"; который можно использовать прямо в тексте.

Ссылки

Ссылки позволяют ссылаться на внешние ресурсы. Например, официальный сайт Go или MDN Web Docs.
